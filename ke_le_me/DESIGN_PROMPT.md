# 渴了么 · 设计 Prompt

## 设计方向
**清新水滴风 · 轻盈健康感**

## Aesthetic Direction
「呼吸感极强的白色空间 + 天蓝色水滴点缀」——
像一杯清水，纯净透明，无多余装饰，让人看一眼就想喝水。

## Color Palette
| 变量 | 色值 | 用途 |
|------|------|------|
| `bgMain` | `#F5F8FF` | 主背景（极浅蓝白） |
| `bgCard` | `#FFFFFF` | 卡片白 |
| `bgSection` | `#EFF4FB` | 分区背景 |
| `blue` | `#29B6F6` | 主色（天蓝水滴） |
| `blueDark` | `#0288D1` | 深蓝强调 |
| `blueLight` | `#E3F2FD` | 浅蓝填充 |
| `green` | `#4CAF50` | 成功/完成 |
| `orange` | `#FF9800` | 警告/未完成 |
| `textPrimary` | `#1A2340` | 主文字 |
| `textSecondary` | `#546E7A` | 次文字 |
| `textHint` | `#90A4AE` | 占位/弱文字 |
| `divider` | `#E8EFF5` | 分割线 |

## Typography
- 主字体: `Noto Sans SC`（中文优先）
- 数字: `DM Mono` 或 `Space Mono`（等宽清晰）
- 标题: 700 weight，主文字色
- 正文: 400-500 weight，次文字色

## Spatial Composition
- 大量留白，内容呼吸感强
- 卡片圆角 16px，阴影极淡（`0 2px 12px rgba(0,0,0,0.06)`）
- 行间距宽松，点击区域 ≥ 48px
- 列表项：左图标 + 中间文字 + 右箭头/数值，iOS 风格

## Key Components

### 进度圆环
- 浅蓝底环 + 渐变蓝色填充（浅→深）
- 中间大号数字，Mono 字体，蓝色
- 下方小字显示百分比

### 卡片
- 纯白背景，极淡阴影
- 顶部无色条（vs 深色主题有色条）
- 圆角 16px

### 按钮
- 主按钮：天蓝色填充，白色文字，圆角 14px
- 次按钮：浅蓝背景（`#E3F2FD`），蓝色文字
- 全宽按钮有轻微阴影

### Toggle / Switch
- 开：天蓝色轨道
- 关：淡灰色轨道
- 滑块：纯白

### 列表行（Settings Row）
- 高度 54px
- 左侧：彩色圆形图标背景（每个功能不同颜色）
- 右侧：灰色文字 + `>` 箭头
- 底部：1px 浅灰分割线

### 底部导航
- 白色背景，顶部淡阴影
- 激活 Tab：天蓝色图标 + 蓝色文字
- 非激活：灰色

## Motion & Animation
- 圆环进度：1.2s ease-out 动画
- 页面切换：向上滑动（200ms）
- 打卡成功：水滴涟漪动画 + 轻微震动反馈
- 开关切换：弹性动画（spring curve）

## Overall Feel
干净、清爽、健康、现代。
像一款被苹果设计师设计的健康类 App。
用户每次打开都感到愉悦，想多喝水。
